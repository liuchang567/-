# 2：菜单上应用/取消应用边框

先给每一个菜单项添加边框;最后一个元素

nav li{
    border-right: 1px solid #666;
}
nav li:last-child{
    border-right: none;
}

也可以直接使用 :not() 伪类来应用元素

nav li:not(:last-child){

}

如果元素有兄弟元素的话，也可以使用通用的兄弟选择符( ~ ）

nav li:first-child ~ li{
    border-left: 1px solid #666
}
